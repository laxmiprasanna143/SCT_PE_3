# SCT_PE_3
Task 03 - Prompting for Task Automation ⚙️

Objective

Use prompting to semi-automate real-world tasks such as summarizing, extracting structured data, or transforming formats.

Prompt

"Convert the following meeting notes into JSON format with keys 'topic' and 'decision'. Ensure the output is valid JSON."

Input-Output Examples

Example 1

Input:

Project deadline extended to October 15
Budget approved for marketing campaign
Next meeting scheduled for Monday
Output:

[
  {"topic": "Project deadline", "decision": "Extended to October 15"},
  {"topic": "Budget", "decision": "Approved for marketing campaign"},
  {"topic": "Next meeting", "decision": "Scheduled for Monday"}
]
